# 音乐个性化推荐系统 python1134

本项目是一个基于Python和Django框架开发的Web项目，采用前后端分离的技术架构。系统主要致力于为用户提供个性化的音乐推荐服务。

## 技术栈

- **开发语言：** Python
- **框架：** Django
- **前端技术：** HTML、CSS、JavaScript、Vue.js
- **后端技术：** Python、Django
- **数据库：** MySQL
- **数据库工具：** Navicat
- **开发软件：** PyCharm
- **版本控制：** Git
- **浏览器兼容：** 谷歌浏览器、火狐浏览器

## 功能模块

### 系统角色

- 管理员
- 用户

### 前台系统功能模块

- **首页界面：** 用户可以看到导航条，并进入各个功能展示页面进行操作。
- **用户注册：** 用户在前端填写信息，后端检查用户名唯一性，并存入MySQL数据库。
- **音乐信息查询：** 用户可查询音乐信息，并进行收藏、点赞或评论。
- **歌曲数据页面：** 用户可查询歌曲数据，并进行收藏、点赞或评论。
- **个人中心：** 用户可进行个人中心、修改密码、我的收藏等操作。

### 后台管理员功能模块

- 管理员可以进行用户管理、音乐分类管理、音乐信息管理、歌曲数据管理、系统管理、个人信息管理等操作。

### 后台用户功能模块

- 用户可以在后台进行个人中心、修改密码、我的收藏等功能的操作。

## 系统运行环境

项目在Windows环境下完美运行。

## 目录结构

具体的目录结构如下：

```plaintext
├── app/                  # 应用代码目录
│   ├── frontend/         # 前端代码目录
│   └── backend/          # 后端代码目录
│       ├── admin/        # 管理员后台目录
│       ├── users/        # 用户相关功能目录
│       ├── music/        # 音乐信息处理目录
│       └── ...
├── db/                   # 数据库文件目录
├── static/               # 静态文件目录
└── ...
```

## 核心亮点

- **个性化推荐：** 根据用户喜好提供个性化音乐推荐。
- **前后端分离：** 前端使用Vue.js，后端使用Django，使得系统结构更加清晰，便于维护。
- **灵活的系统管理：** 管理员与用户功能模块分明，易于管理与使用。
